오 마이 갓! 당신이 영화속 주인공이라면 우리나라에 3% 밖에 없는 라이온킹의 티몬이군요!
자신 또는 친구가 바라본 당신은 현실적이지만 낙천적이고 타인에 대해 긍적적이며 수용적인 모습으로 어떤 모임에서도 잘 적응하는 사람이군요! 친구들이 매우 좋아하는 성격의 소유자로 당신에 주변에는 항상 멋진 사람들이 넘칩니다. 인간애가 많아 모임이나 만나자는 약속을 잘 거절하지도 못하죠!
성격유형 : “자유로운 영혼의 연예인”
티몬 타입의 사람은 이성적이지만 현실을 낙관하는 경향이 큽니다. 타인에 대해 관심이 많으며 때로는 조금 수다스럽기도 하죠. 모임이나 파티를 즐기고 사랑을 받는것, 사랑을 주는 것 모두 좋아하죠.
또한 천부적으로 예술가적 기질이 높은 사람이 많습니다. 자신을 매력적으로 꾸미는데 익숙하며 타고난 미적감각으로 주변을 아름답게 꾸미는 능력이 탁월하죠.
타인에게 사랑받는 것, 타인을 사랑하는 것 모두 좋아하는 당신! 친구분은 이런 당신의 매력에 흠뻑 빠져있을 수 있습니다!
타고난 예술성과 미적인 감각. 누군가에게 주목받는 것, 누군가를 주목하는 것 모두 즐기는 슈퍼 인싸!
하지만 타고난 미적감각과 주변사람들의 인기로 매력적인 본인에게 흠뻑 빠져 지나친 자기애에 빠져 있을 수 있습니다. 또 타인에게 닥친 문제에는 따듯한 위로와 현실적인 조언에 능통하지만 본인의 문제에 대해서는 항상 낙관하는 마음으로 현실을 도피하는 모습을 보일 수 도 있죠.
그러나 티몬과 같이 긍정적이고 인간애가 넘치며 타고난 예술적 감각으로 매력이 넘치는 당신은 항상 주변에 좋은 인연과 친구들을 불러오죠! 지금처럼 밝고 멋진 모습으로 삶을 즐기세요!
